## Description
Create `tools/codegen/studio_codegen.py` with functions: gen_agent_md(role) -> str, gen_command_md(command) -> str, gen_hooks_json(hooks) -> str, gen_template_md(template) -> str, and codegen_studio(ast, out_dir) orchestrator. gen_agent_md produces Claude Code agent .md files with YAML frontmatter. gen_command_md produces slash-command .md files. gen_hooks_json produces a settings.json fragment. gen_template_md produces template skeletons with required sections.

## Acceptance Criteria
- gen_agent_md() produces valid agent .md with name, description, model (tier-based), tools list
- gen_command_md() produces valid command .md with phase, role, prompt, output
- gen_hooks_json() produces valid JSON with hook configurations
- gen_template_md() produces valid template skeleton with all required sections
- codegen_studio() writes all output files to specified directory structure
