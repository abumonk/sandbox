## Description
Create `tools/verify/studio_verify.py` with `verify_escalation_acyclicity()` function. Build a directed graph from role.escalates_to edges. Use Z3 Int ordinals to prove no cycles exist (escalates_to always goes to a strictly higher ordinal). Report all roles in a cycle if one is found. Integrate into ark_verify.py's verify flow — when studio items are present, call studio verification.

## Acceptance Criteria
- Detects cycles in escalation graph and reports all roles in the cycle
- Passes for valid acyclic hierarchies (e.g., Specialist -> Lead -> Director)
- Returns VerifyResult dicts consistent with existing verify output format
- Integration with ark_verify.py: studio checks run when role/studio items present
- Handles edge cases: roles with no escalates_to (top of chain), single-role studios
