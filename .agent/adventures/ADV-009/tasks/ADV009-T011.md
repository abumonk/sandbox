## Description
Remove v1-only code paths that the four new tabs replaced: `renderFileBrowser` for designs/plans/research, legacy `renderPermissions`, legacy `renderReviews`, `renderKnowledge` (moved to Decisions), `renderLog` (moved into disclosures), and any now-dead CSS (legacy `.split` if unused). Keep `parseKnowledgeSuggestions` (still used). Preserve the switchTab fallback for unknown keys.

## Acceptance Criteria
- [ ] No references remain to the removed renderers.
- [ ] `grep -c "renderFileBrowser\|renderKnowledge\|renderLog\|renderPermissions\|renderReviews" index.html` is 0 (outside of commit-message/log strings).
- [ ] Page still loads and all four tabs (Overview, Tasks, Documents, Decisions) work with no console errors.
- [ ] `parseKnowledgeSuggestions` is still present and referenced by `renderDecisions`.
- [ ] `switchTab` coerces unknown keys to `overview` (fallback preserved).
- [ ] Legacy CSS rules `.split`, `.split .file-list`, `.file-list .file-entry*` are removed.
- [ ] `.file-view` and `.kb-*` CSS rules are removed **only if** no v2 renderer still emits those classes (verified by grep before deletion).

## Design
Summary: Surgical post-T010 cleanup of `index.html`. Delete five legacy renderer methods (`renderFileBrowser`, `renderPermissions`, `renderReviews`, `renderKnowledge`, `renderLog`), prune the seven obsolete tab-dispatch branches in `renderTab()`, and remove dead CSS (`.split` family unconditionally; `.file-view` / `.kb-*` only if no JS caller remains). Keep `parseKnowledgeSuggestions` (reused by Decisions), keep `.log-tail` CSS (reused in disclosures), and preserve the `switchTab` unknown-key fallback to Overview. Verify with grep and a four-tab smoke test; expect ~200 lines net deletion.

Full design: [`.agent/designs/ADV009-T011-design.md`](../../../designs/ADV009-T011-design.md)

Key risks: false-positive deletion of `.file-view` / `.kb-*` rules that v2 renderers still use (mitigated by grep-before-delete); stale `'reviews'`-tab link in Overview's adventure-report button (fix forward to `'documents'` if T007 didn't); scope creep (pure removal task).
