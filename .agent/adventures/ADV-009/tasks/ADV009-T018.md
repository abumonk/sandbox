## Description
Extend `server.py` with two new endpoints:

1. `GET /api/adventures/{id}/graph` — imports
   `adventure_pipeline.tools.ir.extract_ir` and serves the
   node/edge/explanations payload specified in
   `design-pipeline-graph-view.md`.
2. `POST /api/adventures/{id}/tasks/{task_id}/depends_on` — detailed in
   `design-graph-edit-affordances.md`; uses a shared `_cycle_free` helper
   to reject self-cycles and cycle-creating edges.

Stdlib only. No changes to the T003/T004 endpoints. Follow existing
`server.py` patterns for route dispatch, frontmatter rewrite, and error
responses.
