## Design

Author 8 files in a new sibling package `adventure_pipeline/` — no edits
to `ark/`. Follow the ADV-008 `shape_grammar/` architectural precedent:
each `.ark` file parses as a standalone unit (re-declaring enums as
needed); cross-file unification happens later in the IR extractor (T016).

**Files and their role:**
1. `adventure_pipeline/__init__.py` — empty package marker.
2. `adventure_pipeline/README.md` — 1-page overview with `## Purpose`,
   `## Layout`, `## Usage`, and `## Deferred invariants` sections.
3. `specs/adventure.ark` — 7 enums (`State`, `Status`, `TCStatus`,
   `ProofMethod`, `DecisionKind`, `DocKind`, `PermCategory`) and 11 entity
   declarations (`Adventure`, `Phase`, `Wave`, `Task`, `Document` +
   4 subtypes, `TargetCondition`, `Decision`, `Permission`, `Agent`,
   `Role`). List-of-id fields typed as `String` per `Task.repos_json`
   precedent; every `class` gets a stub `#process[strategy: code]{ description: ... }`.
4. `specs/pipeline.ark` — 3 state-machine classes
   (`AdventureStateMachine`, `TaskLifecycle`, `ReviewPipeline`) each
   with a `#process` block encoding transitions via `pre:` guards and
   `post:` disjunctions over enum variants; absorbing-terminal temporal
   invariants on `completed`/`cancelled`/`done`.
5. `specs/entities.ark` — 5 runtime classes (`RunningAgent`,
   `ActiveTask`, `PendingDecision`, `KnowledgeSuggestion`,
   `ReviewArtifact`) plus re-declared `Verdict` / `DecisionKind` enums.
6. `specs/verify/state_transitions.ark` — `verify` block with
   terminal-absorbing and blocked-reentry checks.
7. `specs/verify/permission_coverage.ark` — `verify` block;
   cross-entity existentials deferred to README if unexpressible.
8. `specs/verify/tc_traceability.ark` — `verify` block; same deferral
   protocol.

**Validation (before coder marks T015 done):**
- `python ark/ark.py parse adventure_pipeline/specs/{adventure,pipeline,entities}.ark` → all exit 0.
- `git status -- ark/` → clean (no Ark patches).

If Ark's grammar rejects any invariant, document in README "Deferred

IR-roundtrip checks belong to T017 / T016, not T015.
