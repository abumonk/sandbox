## Description
Extend ark_verify.py translate_expr dispatch to handle pipe and param_ref kinds. Pipe: recursively translate head, fold stages. ParamRef: delegate to SymbolTable.

## Acceptance Criteria
- Existing verify tests still pass (`test_verify_full_expr.py`, `test_verify_bridges.py`, `test_verify_temporal.py`)
- New unit tests cover pipe translation (numeric single-stage, multi-stage, opaque stage)
- New unit tests cover param_ref translation (var, prop, idx, nested)
- Unknown stage name raises ValueError with helpful message
- User-defined expression inlining works via expr_registry
- Recursive expression cycle detection raises ValueError

## Design

**Design doc**: [ADV001-T012-design.md](../designs/ADV001-T012-design.md)

**Summary**: Extend `translate_expr` in `ark_verify.py` with two new `elif` branches for `kind == "pipe"` and `kind == "param_ref"`. Add `expr_registry` optional parameter threaded through `translate_expr` for user-defined expression inlining.

**Pipe translation**: Recursively translate `head` via `translate_expr`, then fold stages left-to-right via `apply_stage`. Each stage resolves in priority order: (1) `NATIVE_PRIMITIVES` dict for direct Z3 ops (add, sub, mul, div, abs, clamp-range, str-len), (2) `expr_registry` for user-defined expression inlining with alpha-renaming, (3) `OPAQUE_PRIMITIVES` set for uninterpreted Z3 Functions (str-lower, str-upper, regex, temporal, file-io), (4) unknown stage raises ValueError with candidate list.

**ParamRef translation**: Dispatch on `ref_kind`: `var` delegates to `syms.get(name)`, `prop` joins parts with dots and delegates to `syms.get(path)`, `idx` uses Z3 `Array`/`Select`, `nested` recursively translates inner expr.

**Backward compat**: All new parameters default to `None`. Existing callers unchanged. `build_expr_registry` available but not auto-invoked (future integration task).

**Key files**: `ark_verify.py` (extend translate_expr, add 5 helper functions, add NATIVE_PRIMITIVES/OPAQUE_PRIMITIVES dicts, add z3 imports for If/Length/Function/Array/Select/sorts), `test_verify_expression.py` (NEW, 10 test cases).
