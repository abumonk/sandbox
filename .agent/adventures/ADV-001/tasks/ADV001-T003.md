## Description
Edit lib.rs to add new Expr variants, RefKind, PipeStage, ExpressionDef, PredicateDef, new Item variants, expression_index and predicate_index on ArkFile. Ensure Serialize/Deserialize still work.

## Acceptance Criteria
- cargo build -p ark-dsl clean
- cargo test -p ark-dsl existing tests pass
- New types derive Debug/Clone/Serialize/Deserialize

## Design

Design doc: `.agent/adventures/ADV-001/designs/ADV001-T003-design.md`

**Summary**: Add 5 new types and extend 3 existing types in `dsl/src/lib.rs`:

1. **New types**: `RefKind` (enum: Var/Prop/Idx/Nested), `PipeStage` (struct: name + args), `ExpressionDef` (struct: name/inputs/output/chain/description), `PredicateDef` (struct: name/inputs/check/description)
2. **Extended `Expr`**: Add `Pipe { head, stages }` and `ParamRef { kind, name, path, index, nested }` variants
3. **Extended `Item`**: Add `Expression(ExpressionDef)` and `Predicate(PredicateDef)` variants
4. **Extended `ArkFile`**: Add `expression_index: BTreeMap<String, usize>` and `predicate_index: BTreeMap<String, usize>` with `#[serde(default)]`; add `expression(&self, name)` and `predicate(&self, name)` accessor methods
5. **`build_indices` in parse.rs**: Extend to populate the two new indices

All types derive `Debug, Clone, Serialize, Deserialize`. 10 implementation steps, see design doc for details.
