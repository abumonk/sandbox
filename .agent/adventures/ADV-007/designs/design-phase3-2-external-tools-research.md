# Phase 3.2: External Tools Research - Design

## Overview
Research all external tools and repositories referenced in the roadmap to understand their capabilities, architecture, and integration potential with the Claudovka ecosystem.

## Tools to Research
1. **QMD** - Quality metrics/documentation tool
2. **CodeGraphContext (CGC)** - Code graph context for AI agents
3. **Everything Claude Code** - Claude Code ecosystem/community resource
4. **Claude Code Game Studios** - Game development with Claude Code
5. **LSP Plugins** - Language Server Protocol integrations
6. **MCP Servers**:
   - github, memory, firecrawl, supabase
   - sequential-thinking, vercel, railway, cloudflare
   - clickhouse, ableton, magic
7. **Agent Orchestrator** - Multi-agent orchestration patterns

## Approach
For each tool/repo:
1. Web search to find repository, documentation, community
2. Analyze architecture and key features
3. Assess integration potential with Claudovka ecosystem
4. Identify which roadmap phases each tool serves
5. Rate: must-have / nice-to-have / reference-only
