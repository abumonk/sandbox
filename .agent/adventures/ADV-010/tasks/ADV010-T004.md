## Description
Implement `.agent/telemetry/cost_model.py`: hand-rolled YAML frontmatter reader for `.agent/config.md`, `normalize_model`, `known_models`, `cost_for`. Raise `UnknownModelError` on unknown model IDs. Memoise `load_rates`.

## Acceptance Criteria
- `load_rates()` returns `{"opus":0.015, "sonnet":0.003, "haiku":0.001}` from current config.
- `cost_for("opus", 85000, 28000)` returns 1.695 (= 113 × 0.015).
- `cost_for("unknown", 1, 1)` raises `UnknownModelError`.
- `normalize_model("claude-opus-4-6") == "opus"`.

## Design

Full design: [ADV010-T004-design.md](../designs/ADV010-T004-design.md)

**Approach**: Create three new files under `.agent/telemetry/` — a package init, a shared exception hierarchy (`errors.py` with 7 exception classes rooted at `CaptureError`), and the cost model module (`cost_model.py`). The cost model uses a hand-rolled YAML-subset frontmatter parser (~40 LOC) to read rates from `.agent/config.md`, stdlib only (no PyYAML). Four public functions: `load_rates` (memoised via `lru_cache`), `normalize_model` (regex-based alias resolution), `known_models` (frozen set of rate keys + aliases), and `cost_for` (flat-rate formula: `(in+out)/1000 * rate`, rounded to 4dp, raises `UnknownModelError` on miss).

**Implementation steps** (7 steps):
1. Create `.agent/telemetry/__init__.py` (empty package init)
2. Create `.agent/telemetry/errors.py` (full exception hierarchy from design-error-isolation.md)
3. Implement `_parse_frontmatter(text)` — line-by-line parser for `---`-delimited YAML subset with 2-space indent tracking
4. Implement `load_rates(config_path=None)` — reads config, extracts `adventure.token_cost_per_1k`, returns `{model: rate}` dict, cached
5. Implement `normalize_model(model_id)` — regex matching `claude-{canon}*` / `{canon}*` to canonical short names
6. Implement `known_models()` — union of rate keys and common aliases
7. Implement `cost_for(model, tokens_in, tokens_out, rates=None)` — normalize, lookup, compute, round

**Target files**: `__init__.py`, `errors.py`, `cost_model.py` (all under `.agent/telemetry/`)
