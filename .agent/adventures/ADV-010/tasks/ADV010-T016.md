## Design

Create 11 test files (9 test modules + conftest.py + __init__.py) under `.agent/adventures/ADV-010/tests/` using stdlib `unittest` only. Each test module maps to TC-to-function tables from `design-test-strategy.md`. Tests import from `.agent/telemetry/*` modules. File-writing tests use `tempfile.TemporaryDirectory`. Subprocess tests drive `capture.py` via `subprocess.run` with stdin. Live canary test gates on fixture presence via `@unittest.skipUnless`.

Implementation order: conftest.py shared helpers first, then test_schema -> test_cost_model -> test_capture -> test_aggregator -> test_task_actuals -> test_error_isolation -> test_backfill -> test_live_canary -> test_regression. Checkpoint at 3 green files per the split-if-needed NOTE.
