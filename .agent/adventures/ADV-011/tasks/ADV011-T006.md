## Description
Produce `research/descriptor-delta.md` per section 4 of `design-unified-descriptor.md`. Classify every current `ark/dsl/stdlib/*.ark` file with a verdict from {KEEP-AS-IS, KEEP-RENAMED, MERGE-INTO, MOVE-TO-PRIMITIVES, RETIRE}. Cite every descriptor-bucket concept from the mapping. Cite every dedup row assigned to the descriptor. Include the target two-level stdlib layout, the grammar authoring contract, and the AST family spec. Note: the unified design doc itself already exists under `designs/` — this task produces only the delta report.

## Acceptance Criteria
- File `.agent/adventures/ADV-011/research/descriptor-delta.md` exists.
- Verdict table has exactly 9 data rows (one per `ark/dsl/stdlib/*.ark` file) with six columns (`file`, `verdict`, `target`, `rationale`, `concept_refs`, `dedup_refs`).
- Every `verdict` value is one of `{KEEP-AS-IS, KEEP-RENAMED, MERGE-INTO, MOVE-TO-PRIMITIVES, RETIRE}` (TC-009 basename coverage for `types`, `expression`, `predicate`, `code_graph`, `code_graph_queries`, `studio`, `evolution`, `agent`, `visual`).
- Every row from `research/concept-mapping.md` with `bucket = descriptor` is cited in section 5a.
- Every row from `research/deduplication-matrix.md` with `assigned_bucket = descriptor` is cited in section 5b.
- Section 5c contains the literal strings `ADV-008` and `host-language` so TC-010's grep passes.
- Document contains five H2 sections: Verdict Table, Target Two-Level Stdlib Layout, Grammar Authoring Contract, AST Family Spec, Citations.
- Grammar Authoring Contract declares Lark primary / Pest secondary with a grammar-parity autotest.
- AST Family Spec names a single `Item` variant enum with ~40 variants grouped by domain.

## Design
Full design: [`.agent/designs/ADV011-T006-design.md`](../../../designs/ADV011-T006-design.md)

**Summary.** The deliverable is a single research artefact,
`research/descriptor-delta.md`, with five H2 sections: (1) a 9-row verdict
table over every `ark/dsl/stdlib/*.ark` file using the six-column schema
`file | verdict | target | rationale | concept_refs | dedup_refs` and an
ordered first-match decision tree over the five verdicts
`{MOVE-TO-PRIMITIVES, RETIRE, MERGE-INTO, KEEP-RENAMED, KEEP-AS-IS}`; (2) the
target two-level layout tree (`primitives/` + `domain/`) from §4 of
`design-unified-descriptor.md`, annotated with verdict-row references; (3) a
grammar authoring contract making Lark primary and Pest a byte-identical
mirror, enforced by a grammar-parity autotest and a one-patch rule across
both files; (4) the AST family spec — single `Item` variant enum in
`ark-dsl` (Rust) and `ark_parser.Ast` (Python) with ~40 seed variants grouped
into base/primitives/studio/evolution/agent/visual/code_graph buckets; (5) a
citation manifest covering every descriptor-bucket concept-mapping row,
every descriptor-bucket dedup row, and a prose citation of ADV-008's
host-language feasibility study (the literal strings `ADV-008` and
`host-language` satisfy TC-010). Seed verdict guidance: `types.ark`,
`predicate.ark`, `expression.ark` → MOVE-TO-PRIMITIVES;
`code_graph_queries.ark` → MERGE-INTO `domain/code_graph.ark#queries`;
remaining five domain files → KEEP-RENAMED into `domain/`.
